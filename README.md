# mharj-temperature
Temperature Web element

### Bower setup
```json
  "dependencies": {
    
    "mharj-temperature": "https://github.com/mharj/mharj-temperature.git"

  }
```
### Options
```javascript
  element.value = 20;
```
### Html
```html
  <mharj-temperature style='height: 200px' value='-30'></mharj-temperature>
```
